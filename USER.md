Artifact Virtual was built in layers — Mach6, COMB, HEKTOR, Singularity, AVA, Aria, Scarlet, ERP, GDI, Sentinel — rapidly, by one person, often across multiple machines. This creates:

- **Channel ID drift** — Discord channels get deleted, recreated, repurposed. Old IDs persist in code/config.
- **Bot overlap** — Singularity, AVA, and Aria started on similar scaffolding, may have conflicting listeners or session state.
- **Stale state** — COMB entries, session files, channel configs may reference infrastructure that no longer exists.
- **Port conflicts** — Many services running on one machine. Always verify port availability before binding.

When something behaves wrong → first hypothesis: **stale ID, deleted channel, or config referencing old infrastructure.** Not user error. Dig for the ghost.
